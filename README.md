### Vehicles

<em> Introducción <em>

Para la elavoracion de este proyecto, se creo un entorno virtual, en el cual se instalaron las librerias pandas, plotly_express y streamlit, para poder cargae el dataframe, poder realizar las graficas y realizar la aploicacion.

<em> Material <em>

El dataframe que se ocupo es de una base de datos de vehiculos, la informacion que contiene en general es el precio de los vehiculos, el año, el kilometraje (en millas), el precio, informacion mas detallada (tipo de combustible, cilindrada, etc) continuacion:

<em> Procedimiento <em>

Las columnas que se utilizaron para la creacion de la app fueron la columna de price, odometer, para el histograma se hizo una suma de vehiculos vendidos de acuerdo a su kilometraje, para el diagrama de dispersion en base al kilometraje del carro, asi como las condicones del mismo, se le asigno el precio, este se utilizo para el diagrama.

Se creo un repositorio en github y se clono en la computadora, al elaborar todos los archivos requeridos, por medio de una sincronizacion entre VS code y github se subio al repositoria, por medio de render podemos compartir dicha app en internet para que cualquier usuario la pueda utilizar desde su computadora.

<em> Conclisión <em>

La aplicacion que creamos con streamlit nos permite crear un histograma que nos muestra las unidades de vehiculos vendidos de acuerdo a las millas recorridas. Al mismo tiempo nos permite crear un diagrama de dispersion de la unidad en base a sus milllas recorridas.
